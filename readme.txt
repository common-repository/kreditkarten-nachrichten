== Description ==

Das Kreditkarten-Nachrichten-Plugin stellt einen Nachrichten-Ticker aus dem Finanzbereich von [http://www.kreditkarte.org/](http://www.kreditkarte.org/) zur Verf&uuml;gung. Das Plugin ist direkt im widget-Bereich konfigurierbar.

== Installation ==

1. Extract `kreditkarten-nachrichten.zip`.
2. Upload the `kreditkarten-nachrichten`-folder to your `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Add the widget to your sidebar from Appearance->Widgets and configure the widget options.
